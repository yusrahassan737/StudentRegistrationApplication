# Student Registration Application

## Scenario
A small college would like a simple student registration application.
The application consists of two pages:
**Page 1: Student Registration**
Students complete a registration form.
When submitted, the information is stored in a MySQL database.
**Page 2: Student Directory**
Retrieve all students from the database
and display them in React.

## Tools
• React
• PHP (using Apache service from XAMPP)
• MySQL (using XAMPP)
• HTML
• CSS

## Instructions
To run the back-end:
- Clone this repository into 'C:\xampp\htdocs\'.
- Start Apache and MySQL in XAMPP.
- Run SQL
To run the front-end:
- navigate to the frontend folder
- in the terminal type "npm run dev".
- open [localhost](http://localhost:5173/)
